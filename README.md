## Overview

This project simulates multipath propagation for passive radar using GPU-accelerated ray tracing.

Key features:
- Static multipath field generation (TX → RX)
- Efficient caching of ray-traced fields
- 3D visualization of propagation paths
- Designed for UAV detection workflows
